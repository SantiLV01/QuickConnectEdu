# QuickConnectEdu

Plataforma educativa desarrollada como proyecto académico, enfocada en facilitar la conexión y comunicación rápida entre estudiantes y docentes.

## Tecnologías utilizadas
- **Java**
- **JavaFX**
- **FXML**
- **CSS (JavaFX Styles)**
- **Scene Builder**
- **NetBeans IDE**
- **Git**
- **GitHub**

## Funcionalidades
- Comunicación rápida entre estudiantes y docentes
- Interfaz gráfica intuitiva y fácil de usar
- Organización de contactos o secciones educativas
- Navegación fluida entre módulos

## Objetivo del proyecto
Mejorar la comunicación en entornos educativos mediante una aplicación sencilla, visual y accesible para los usuarios.
